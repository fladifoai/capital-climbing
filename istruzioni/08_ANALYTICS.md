# Analisi e dashboard

## KPI

- vie completate;
- punti totali;
- grado massimo;
- massimo On-sight;
- massimo Flash;
- percentuale OS + Flash;
- falesie visitate;
- progetti attivi;
- sessioni;
- tentativi.

## Grafici MVP

1. progressione dei gradi;
2. distribuzione per grado;
3. punteggio cumulativo;
4. vie per anno;
5. tipi di salita;
6. punti per falesia.

## Filtri

- anno;
- falesia;
- grado;
- tentativo;
- pubblico/privato nella vista personale.

## Profilo pubblico

Le analisi del profilo pubblico devono usare soltanto:

```text
visibility = public
```
