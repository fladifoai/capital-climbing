## Comandi utili

```bash
# Dry run (nessuna richiesta)
python3.13 run.py --dry-run

# Test singola falesia
python3.13 run.py --crag "Collepardo"

# Run completo con cache
python3.13 run.py --use-cache

# Re-scarica tutto senza cache
python3.13 run.py --refresh-cache

# Limita a N falesie
python3.13 run.py --max-crags 5
```
